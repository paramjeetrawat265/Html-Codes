b tag

The b tag in HTML is used to specify the bold text without any extra importance. The text is written within b Html tag display in bold size. You can do that by using font-weight: bold; property in CSS. It is a container tag that contains an opening tag, content & closing tag

i tag

The "i" tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. The "i" tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.
